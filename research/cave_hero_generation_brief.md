# CAVE 实验室真实资料驱动主视觉规范

## 目的

进入页右侧图像应让熟悉 UCL CAVE 的研究人员立即识别出这是一个**大型、明亮、可重构的室内空气质量与通风实验大厅**，而不是黑色科幻舱、洁净室或数据中心。图像用于网站 hero，左侧已有标题与操作入口，因此主体集中在画面右侧和中部，左缘保持较暗、低信息密度。

## 已核验的真实空间特征

| 特征 | 视觉表达 |
| --- | --- |
| 大厅尺度 | 约 204–206 m²、9–9.54 m 高，呈高挑矩形工业大厅 |
| 围护 | 高大哑光白色保温墙板，竖向分缝，浅灰耐磨地面 |
| 服务系统 | 暴露镀锌金属导管、少量矩形风口、墙面检修板和工业门 |
| 低位构件 | 长条银灰色低位送回风/服务组件，端部有安全黄色保护件 |
| 高低位风路 | 高、低位置均可送风与排风；以少量冷蓝/暖红透明气流表现 |
| 实验中建筑 | 大厅内部可搭建一层或两层临时全尺度测试房间，独立控制内外环境 |
| 测量系统 | 墙面竖向温湿度测点阵列；可表现少量小型白色/灰色数据记录器与细线标记 |
| 颜色 | 主体白、浅灰、镀锌银；安全黄仅用于保护端部；蓝/红仅编码冷热送风 |

## 构图与风格

采用从大厅入口或略高观察点看向内部的三分之四透视。画面中放置一个朴素的、可拆装的一层测试房间：白色板墙、普通门窗、简洁金属框架；不要玻璃舱、发光透明墙或未来主义机器。上方留出真实的高空间感，墙面显示高低位风口和竖向测点阵列。光线为均匀明亮的实验设施照明，允许网站通过深色遮罩收束为现有黑色进入页。

## 生成图中不直接渲染的内容

尺寸线、测点编号、空气流向标签与实验阶段信息由网页 HTML/SVG 叠层精确呈现，生成图本身不含文字、数字、徽标或 UI。

## 必须避免

避免黑色电影布景、地下掩体、圆形舱门、霓虹灯、密集天花风管、机械臂、人体模型、服务器机柜、全玻璃实验舱、烟雾弥漫、危险品场景、无法施工的建筑结构，以及把 CAVE 误画成虚拟现实 CAVE。
